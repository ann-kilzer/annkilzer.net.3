## How to run it locally

Install dependencies (and set up Husky)
```sh
npm i
```

Run in dev mode
```sh
npm run dev
```

## Quality

Run unit tests
```sh
npm test
```

Run E2E tests
```sh
npm run e2e
npm run e2e:ui
```
